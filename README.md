# sourceshop



## 简介

### 🙂🙂🙂项目介绍

用bootstrap写的响应式购机商城

在线演示：[https://bsshop.imold.wang](https://bsshop.imold.wang)



### 👍👍👍项目特点

- 响应式网页，适配于各种屏幕尺寸的设备
- 多种网页动态效果，增强用户体验
- 简洁清新的网页设置，赏心悦目
